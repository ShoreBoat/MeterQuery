# 电表监控 App

复刻小程序与 DL/T 645 电表的蓝牙通讯，独立查询剩余电量并在低于阈值时发系统通知。

## 在线打包（无需本地装任何东西）

1. 在 GitHub 新建一个仓库（Public 或 Private 都行）
2. 把本文件夹所有内容上传到仓库（网页可直接拖拽，注意要包含隐藏的 `.github` 文件夹）
3. 进仓库的 **Actions** 标签页，如果提示启用 workflow 就启用
4. 每次 push 会自动编译；也可在 Actions 页点 **Build APK → Run workflow** 手动触发
5. 编译完成后，进入那次运行记录，页面底部 **Artifacts** 里下载 `meter-app-debug`
6. 解压得到 `app-debug.apk`，传到安卓手机安装（需允许"未知来源"）

## 使用

1. 打开 App，授予蓝牙、定位、通知权限
2. 确认电表地址和名字关键字（默认已填）
3. **先把微信小程序彻底退出**（否则表被独占，扫不到）
4. 手机靠近电表，点"立即查询一次"
5. 看"解析结果"里是否出现原始帧和数据
6. 调好阈值和间隔，点"开始定时监控"

## 注意

- 剩余电量字段目前是多候选值（leftKwhCandidates），需要你对照表上真实读数确认哪个候选/偏移正确后，告诉开发者改成精确字段，阈值判断才准确。
- App 在前台时定时监控有效；长期后台需要额外做前台 Service（第二版）。
